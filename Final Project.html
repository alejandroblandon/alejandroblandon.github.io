Final Project

Hello! For my final program I decided to create a trivia game similar to the game show "Who Wants to Be a Millionaire". It features a series of 10 random multiple choice questions in which each correct answer will accumulate $100,000 in REAL MONEY!!! Although it sounds simple, there are many cool features to the game such as score trackers, many interactive buttons, a character select screen (purely aesthetic), and even an audience. Hope you enjoy!

HOW TO PLAY:

  1. Hit "Start!" button
  2. Select your character
  3. Click on desired answer
  4. Win!
